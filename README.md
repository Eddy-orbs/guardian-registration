## Run with local ganache
### Prepare the local ganche network
1. `cd ganache-env`
2. `npm run start-ganache` (This will start a prcess that will block the terminal)
3. `npm run compile-and-deploy`
### Start the app
1. `npm run start`



## Available Scripts
